 🔧 TallerApp — Sistema de Gestión de Taller

> **Proyecto académico **  
> Sistema de gestión para taller de reparación de vehículos.


---

## 🚀 Stack Tecnológico

| Capa | Tecnología |
|------|-----------|
| **Backend** | Node.js + Express 5 + MySQL2 (Pool) |
| **ORM** | Prisma (MariaDB/MySQL) |
| **Auth** | JWT + bcrypt |
| **BD** | MySQL 8 / MariaDB — `tallerapp_db` |
